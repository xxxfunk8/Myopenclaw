# 护理助手数据下载系统

## 系统概述

这是一个专为护理助手系统设计的数据下载和管理系统，能够定时登录系统并下载PDF格式的数据报告。

## 功能特性

- 用户认证和登录管理
- 定时任务调度
- PDF文件下载和管理
- 完整的Web界面控制面板

## 技术栈

- PHP (后端逻辑)
- HTML/CSS/JavaScript (前端界面)
- JSON (数据交换格式)

## 文件结构

```
login_system/
├── index.php                 # 登录界面
├── dashboard.php            # 控制面板主界面
├── login_handler.php        # 登录处理逻辑
├── scheduler.php            # 定时任务调度器
├── pdf_downloader.php       # PDF下载处理器
├── downloads/               # 下载文件存储目录
└── README.md               # 本说明文件
```

## 部署说明

### 环境要求

- Web服务器 (Apache/Nginx)
- PHP 7.4 或更高版本
- 支持cURL扩展
- 可选：支持浏览器自动化的环境（用于实际登录操作）

### 基础部署

1. 将整个 `login_system` 目录上传到您的Web服务器
2. 确保 `downloads/` 目录具有写入权限
3. 访问 `dashboard.php` 来使用系统

### 完整功能部署（推荐）

要启用完整的自动登录和PDF下载功能，需要在支持浏览器自动化的环境中部署：

1. 安装Chrome或Chromium浏览器
2. 安装Puppeteer或Selenium WebDriver
3. 修改 `pdf_downloader.php` 中的实际下载逻辑

## 重要说明

由于当前环境限制（如GitHub Codespace），无法直接执行浏览器自动化操作。本系统提供了一套完整的后端接口和前端界面，可在支持完整浏览器功能的环境中部署运行。

## 定时任务配置

系统支持以下频率的定时下载：
- 每日
- 每周
- 每月

定时任务需要通过服务器的cron作业或类似调度机制来触发`scheduler.php`文件。

## 安全考虑

- 登录凭据应安全存储，不应硬编码在生产环境中
- 应使用HTTPS加密传输
- 定期清理下载目录以节省空间

## 故障排除

如果遇到问题：
1. 检查服务器PHP配置和所需扩展
2. 确认目录权限设置正确
3. 查看服务器错误日志
